<p align="justify">
Projeto criado para ajudar equipe de campo das empresas terceirizadas da VIVO. </br>
Aplicação tinha como objetivo fornecer dados dos cabos metálicos utilizados para distribuição de linhas telefônicas e ‘internet’. O usuário fornecia o número do PAR que o cabo iniciava e depois o PAR desejado, com esses dois dados a aplicação consultava toda relação de possibilidade e fornecia as cores dos fios necessárias para identificação. </br>
Essa aplicação poupa muito tempo em pesquisas a tabelas físicas extensas, otimizando assim o tempo e aumento a produtividade em campo.  
</p>
